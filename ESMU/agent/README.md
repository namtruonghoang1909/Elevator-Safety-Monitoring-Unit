# AI Agent Workspace

This directory contains metadata and state files for the Gemini AI agent assisting in the development of the ESMU project.

## Files

- **[CHECKPOINT.md](checkpoint.md)**: Session-to-session state and current focus.
- **[GEMINI.md](GEMINI.md)**: Fundamental mandates and rules for the agent.
- **[TODO.md](todo.md)**: Interactive task tracking.
- **[MEMORY.md](memory.md)**: Long-term record of architectural decisions and completed features.
- **[TASK.md](task.md)**: Current active user request details.
- **[ROADMAP.md](roadmap.md)**: High-level project goals and phase tracking.

## Purpose

These files allow the AI agent to maintain context, adhere to project-specific coding standards (e.g., `esp_err_t` logging), and systematically execute complex multi-file tasks while keeping the user informed of its progress.
